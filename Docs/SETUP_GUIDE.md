# ⚙️ دليل التثبيت والتشغيل (Setup Guide)

يوضح هذا الدليل الخطوات التفصيلية خطوة بخطوة (Step-by-Step) لإعداد بيئة التطوير المحلية وتشغيل مشروع **طبيبي (Tabibi)** بنجاح على جهازك.

## 📋 المتطلبات البرمجية المسبقة (Prerequisites)
تأكد من تثبيت البرامج التالية على جهازك قبل البدء:
1. **PHP**: الإصدار 8.0 أو أحدث.
2. **قاعدة البيانات**: MySQL 5.7+ أو MariaDB 10.4+.
3. **خادم ويب (Web Server)**: Apache (مع تفعيل `mod_rewrite`) مدمج مع XAMPP، أو Nginx.
4. **Node.js**: الإصدار 18 فما فوق، مع مدير الحزم `npm`.
5. **محرر أكواد**: يُنصح باستخدام Visual Studio Code.

---

## 🛠️ الخطوة 1: استنساخ المشروع وإعداد قاعدة البيانات

1. افتح موجه الأوامر (Terminal) في مجلد العمل الخاص بك:
   ```bash
   git clone <رابط_المستودع>
   cd WebTabibi
   ```
2. قم بفتح `phpMyAdmin` (أو أي أداة إدارة MySQL مثل DBeaver).
3. قم بإنشاء قاعدة بيانات جديدة ويفضل تسميتها `uyyuppcc_DBTabibi` (تأكد أن الـ Charset هو `utf8mb4_unicode_ci` لضمان عمل اللغة العربية بشكل صحيح).
4. قم باستيراد ملف قاعدة البيانات المرفق مع المشروع. عادةً يكون في مجلد `backend/sql` أو الملف الذي يحمل اسم `uyyuppcc_DBTabibi_1_.sql`.

---

## ⚙️ الخطوة 2: إعداد وتشغيل الخادم (Backend - PHP)

الواجهة الخلفية هي قلب النظام، يجب تشغيلها أولاً لكي يستطيع الـ Frontend الاتصال بها.

1. افتح محرر الأكواد على مجلد `backend`.
2. قم بتعديل إعدادات الاتصال بقاعدة البيانات. افتح الملف `config/database.php`:
   ```php
   // داخل ملف backend/config/database.php
   define('DB_HOST', 'localhost');
   define('DB_NAME', 'uyyuppcc_DBTabibi'); // اسم القاعدة
   define('DB_USER', 'root');              // اسم المستخدم (عادة root)
   define('DB_PASS', '');                  // كلمة المرور (فارغة في XAMPP)
   ```
3. لتشغيل السيرفر المحلى، افتح الـ Terminal داخل مجلد `backend` واكتب الأمر التالي:
   ```bash
   php -S localhost:8000
   ```
   > **ملاحظة**: سيبدأ السيرفر بالعمل على المنفذ 8000. إذا كنت تستخدم XAMPP، يمكنك ببساطة وضع مجلد `backend` داخل `htdocs` وسيعمل تلقائياً. تأكد أن رابط الـ API المحلي الخاص بك أصبح: `http://localhost:8000/api`.

---

## 💻 الخطوة 3: إعداد وتشغيل واجهة المستخدم (Frontend - React)

1. افتح نافذة Terminal **جديدة** (واترك السيرفر الأول يعمل).
2. ادخل إلى مجلد `frontend`:
   ```bash
   cd frontend
   ```
3. قم بتثبيت الاعتماديات وحزم الـ Node.js:
   ```bash
   npm install
   ```
4. تأكد من إعدادات الربط مع الخادم المحلي. افتح ملف `vite.config.js` وتأكد من إعدادات الـ Proxy:
   ```javascript
   // يجب أن يكون الـ target يشير إلى المنفذ 8000 الذي قمنا بتشغيله للتو
   proxy: {
     '/api': {
       target: 'http://localhost:8000',
       changeOrigin: true,
     }
   }
   ```
5. قم بتشغيل بيئة التطوير المحلية (Dev Server):
   ```bash
   npm run dev
   ```
6. سيظهر لك رابط (عادة `http://localhost:5173`). قم بفتحه في المتصفح وستظهر لك شاشة المشروع الرئيسية!
